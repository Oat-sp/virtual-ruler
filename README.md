# Virtual Ruler Simulator

เว็บจำลองไม้บรรทัดสำหรับฝึกการอ่านค่าความยาว เหมาะสำหรับใช้ประกอบการสอนเรื่องการวัดในรายวิชาวิทยาศาสตร์หรือคณิตศาสตร์

ออกแบบให้ใช้งานได้ทั้งบนคอมพิวเตอร์และ iPad ผ่านเว็บเบราว์เซอร์

## Features

- สุ่มวัตถุสำหรับฝึกวัดความยาว
- วัตถุเป็นสี่เหลี่ยมสีฟ้า อ่านง่าย
- มีเส้นประสีแดงจากขอบซ้ายและขอบขวาของวัตถุลงถึงไม้บรรทัด
- เปลี่ยนชนิดไม้บรรทัดได้ โดยวัตถุเดิมไม่เปลี่ยน
  - ไม้บรรทัดปกติ ความละเอียด 0.1 cm
  - ไม้บรรทัดความละเอียด 0.5 cm
  - ไม้บรรทัดความละเอียด 1 cm
- มีขอบว่าง 0.5 cm ก่อนเส้น 0 และหลัง 15 cm
- แสดงเฉลยค่าจริงของวัตถุ
- ปุ่มเฉลยเปลี่ยนเป็น “ซ่อนเฉลย” อัตโนมัติ
- รองรับการใช้งานบน iPad
  - ลากไม้บรรทัดด้วยนิ้วหรือ Apple Pencil
  - ลากพื้นที่ว่างเพื่อ pan
  - pinch zoom ด้วยสองนิ้ว
- รองรับการ zoom ด้วยปุ่ม
- มีโหมดเต็มหน้าจอ

## How to Use

1. กดปุ่ม **สุ่มใหม่** เพื่อสร้างวัตถุใหม่
2. ลากไม้บรรทัดให้ตำแหน่งเหมาะสม
3. ให้นักเรียนอ่านค่าความยาวและตอบปากเปล่า
4. กด **เฉลย** เพื่อแสดงค่าจริง
5. กด **ซ่อนเฉลย** เพื่อซ่อนคำตอบ
6. เปลี่ยนชนิดไม้บรรทัดเพื่อฝึกการประมาณค่าจากความละเอียดที่ต่างกัน

## Controls

| ปุ่ม | หน้าที่ |
|---|---|
| สุ่มใหม่ | สุ่มความยาวของวัตถุใหม่ |
| เฉลย | แสดงค่าจริงของวัตถุ |
| ซ่อนเฉลย | ซ่อนคำตอบ |
| เต็มหน้าจอ | เปิด/ปิดโหมดเต็มหน้าจอ |
| Zoom + | ขยายภาพ |
| Zoom - | ย่อภาพ |
| รีเซ็ต | คืนค่ามุมมองเริ่มต้น |
| ปกติ | ใช้ไม้บรรทัดความละเอียด 0.1 cm |
| 0.5 cm | ใช้ไม้บรรทัดความละเอียด 0.5 cm |
| 1 cm | ใช้ไม้บรรทัดความละเอียด 1 cm |


## Suggested Classroom Use

กิจกรรมที่เหมาะสม:

- ให้นักเรียนอ่านค่าจากไม้บรรทัดปกติ
- เปลี่ยนเป็นไม้บรรทัด 0.5 cm แล้วให้นักเรียนประมาณค่า
- เปลี่ยนเป็นไม้บรรทัด 1 cm แล้วอภิปรายว่าความละเอียดของเครื่องมือมีผลต่อคำตอบอย่างไร
- ให้นักเรียนเปรียบเทียบคำตอบของตนเองกับค่าจริง
- ใช้ถามตอบหน้าชั้นเรียนแบบปากเปล่า
